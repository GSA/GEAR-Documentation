Field reference for Sample API data

The fields below are returned in the Sample API.

Business Application API Fields
Field name Description Data type
Id The database key that identifies the Business Application. string
Name Name of the Business Application. string
Description Description of Business Application. string
SSO Verbose name of the SSO that manages the application. USUALLY the parent SSO is the “Owner,” but not always. string
SSOShort Short name of the SSO that manages the application. USUALLY the parent SSO is the “Owner,” but not always. string
Owner Verbose name of the 2-letter office(s) that manages (co-manage) this application. string
OwnerShort Short name of the 2-letter office(s) that manages (co-manage) this application. string
System The Parent System name that includes this application. If not part of a System, then the value will be NULL. string
BusinessPOC List of business POCs, separated by commas string
TechnicalPOC List of technical POCs, separated by commas string
Cloud Is this Business Application hosted in the Cloud (as opposed to in a GSA Data Center)? Yes or No only. string
TechnologyPlatform Technology platform for the application. One of the following values:
  • .Net
  • Appian
  • ArcGIS
  • Business Objects
  • ColdFusion
  • Django
  • Documentum
  • Drupal
  • Foxpro
  • Google App Engine (GAE)
  • J2EE
  • Microsoft Office
  • Oracle
  • PowerBuilder
  • Provided as Package
  • Red Hat
  • Ruby on Rails
  • Salesforce
  • Unisys Mainframe
  • WordPress
  • Other
 string
Status Status of the application. One of the following values:
  • Pre-Production
  • Production
  • Retired
 string
Alias Other names by which this application is known. string
RegionClassification PBS-centric classification. Values will be “National” or “Regional”. string
HostingProvider Service Provider for the application. One of the following values:
  • Amazon
  • Appian
  • BSP
  • CGI
  • CTAC
  • ESRI
  • GSA
  • Google
  • HP Enterprise Services
  • IBM
  • Marathon Digital Services
  • Monster Govt Services
  • Noblis
  • OPM
  • Salesforce
  • SAP
  • ServiceNow
  • USDA Phase 1
  • USDA Phase 2
  • Vanderweil (VFA)
 string
FismaSystem The FISMA System that provides an ATO for this application. string
Investment The eCPIC Investment that provides funding for this application. string
IsRevenueGenerator Does this Business Application generate revenue? Values limited to Yes, No and TBD string
DesktopComponent Does this Business Application have a component that can be installed on a laptop or desktop? Values limited to Yes, No and TBD. string
RetiredYear Year the application was retired. Will be NULL if “Status” is not “Retired” string
Capabilites A comma separated list of the FEAF BRM capabilities that are supported by this application. string
Technologies A comma separated list of IT Standards that compose this application. string
Link The URL that points to the applications web interface. If there are more than one, they will be separated by commas. string
IT Standards API Fields
Field name Description Data type
Id The database key that identifies the IT Standard. string
Name Name of the IT Standard. string
Description Description of IT Standard. string
Type The type of IT Standard. This value will only be Software. string
Category The category(ies) that apply to this standard. Multiple categories may apply, and will be separated by commas. string
Status The status of the IT Standard. One of the following values:
  • Approved - Preferred
  • Approved - Maintain
  • Pilot
  • Proposed
  • Exception
  • Sunsetting
  • Denied
  • Not Yet Submitted
 string
DeploymentType The typical deployment of an IT Standard. One of the following values:
  • Desktop
  • Server
  • PaaS
  • SaaS
 string
Comments Additional comments to add context to the IT Standard entry. string
POC The point of contact for this IT Standard string
ReferenceDocuments This is a hyperlink to the Request Tracker entry relating to the IT Standard request. string
ApprovalExpirationDate Year and month that the approval for this standard expires. Standards that are not in an "approved" or "pilot" state will have NULL for this field. string
FISMA System API Fields
Field name Description Data type
Id The database key that identifies the FISMA System string
Name Name of the FISMA System. string
RelOrgDisplayName The short name of the organization that is related to this FISMA System. string
FedContractorLoc Is this FISMA System related to a Federal or Contractor location. string
FIPS199 The FIPS 199 impact level of the FISMA System. One of the following values: High, Moderate, or Low. string
ATODate Date was the ATO isssued string
ATOType Type of ATO that certifies this FISMA System. string
RenewalDate Date when the ATO needs to be renewed. string
ComplFISMA Has the annual FISMA review for this FISMA System been completed for this Fiscal Year. string
RelatedArtifacts Letters and other artifacts related to this FISMA System. This will be returned as an array containing zero, one, or many Artifacts, containing the NAME of the Artifact and ReferenceDocuments, which will be the URL of the Artifact. string
FISMASystemIdentifier A shortened identifier for FISMA Systems for uses with Security's Continuous Monitoring System. string
ISSO ISSO for this FISMA System. string
ISSM ISSM for this FISMA System. string
AO Authorizing Official for this FISMA System. string
PM Program Manager for this FISMA System. string
System API Fields
Field name Description Data type
Id The database key that identifies the System string
Name Name of the System. string
Description Description of the System. string
SSO Verbose name of the SSO that manages the System. string
Organization API Fields
Field name Description Data type
Id The database key that identifies the Organization string
Name Name of the Organization. string
Description Description of the Organization. string
Parent Verbose name of the Parent Organization. string
Business Capability API Fields
Field name Description Data type
Id The database key that identifies the Business Capability string
Name Name of the Business Capability. string
Description Description of the Business Capability. string
Parent The parent capability to this Business Capability. string
ReferenceNum The hierarchical reference number for this Business Capability. string
Investment API Fields
Field name Description Data type
Id The database key that identifies the IT Investment. string
Name Name of the IT Investment. string
Description Description of the IT Investment string
Type How this IT Investment is characterized. Values are "Major IT Investment," "Non-Major Investment," or "Funding Transfer Investment" string
PSA The Primary Service Area of this IT Investment. This is the main Business Capability that this investment supports. string
SSA The Secondary Service Area of this IT Investment. This is a secondary Business Capability that this investment may support. string
InvManager The Point of Contact for this IT Investment string
UII The Unique Investment ID for this IT Investment that is tracked in eCPIC. string
